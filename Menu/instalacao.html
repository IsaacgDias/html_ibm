Primeiro passso
Para se preparar para a implantação do sistema Guardium® , o administrador da rede precisa fornecer as seguintes informações.

- Endereço IP para a placa de interface (a interface primária)

- Máscara de sub-rede para endereço IP primário

- Endereço IP do roteador padrão.

- Hostname e nome de domínio para designar ao sistema

- Endereços IP do servidor DNS (até três endereços), e adicionar o novo sistema Guardium ao seu domínio DNS

- (opcional) Endereço IP para interface de gerenciamento secundário

- (opcional) Mask para interface de gerenciamento IP secundário

- (opcional) Gateway para interface de gerenciamento IP secundário

- (opcional) Nome do host do servidor NTP

- (opcional) informações de configuração do SMTP (para alertas de e-mail): endereço IP, porta e se a autenticação for usada, um nome de usuário e senha SMTP

- (opcional) informações de configuração SNMP (para alertas SNMP) o endereço IP do servidor SNMP e o nome da comunidade de trap a utilizar.

Segundo passso
Depois que o aparelho foi carregado no rack do cliente, conecte o aparelho à rede da seguinte maneira:

- Encontre as conexões de energia. Plugue o cabo (s) de alimentação (s) apropriado (s) nessas conexões

- Conecte o cabo de rede à porta de rede a interface primária . Conecte quaisquer cabos de rede secundários opcionais.

- Conecte um Teclado, Video e Mouse diretamente ou através de uma conexão KVM (seja serial ou através da porta USB) para o sistema.

- Power up o sistema.

Instalar a imagem do Guardium.
...

Terceiro passso
Esta seção explica como instalar a imagem e particionar o disco.

- Certifique-se de que as configurações de sua UEFI/BIOS "boot sequence " são definidas para tentar a inicialização a partir da mídia removível (a unidade CD/DVD) antes de usar o disco rígido.

- Carregue a imagem Guardium® a partir do DVD de instalação.

- Aparecem as duas opções a seguir:

Instalação Padrão: esta é a inadimplência. Use esta escolha na maioria dos casos ao particionar o disco. Instalação de Partição Customizada: permite mais customização de todas as partições (localmente ou em um disco SAN). Veja partidarização personalizada para obter mais informações sobre como implementar essa opção.

*Na primeira inicialização após a instalação, o usuário é convidado a aceitar um Contrato de Licenciamento. Eles podem usar PgDn para ler através do acordo ou Q para pular até o final. Para aceitar os termos do acordo, digite q para sair e, em seguida, digite sim. O usuário deve digitar sim para o acordo ou a máquina não irá inicializar.

- O sistema bota para cima a partir de DVD. São necessários cerca de 12 minutes minutos para esta instalação. O processo de instalação irá agora pedir que você escolha um coletor ou agregador (será configurado como "Collector" automaticamente após 10 seconds se nenhuma entrada for fornecida). Veja a Visão Geral do Produto para uma explicação de Collector e Aggregador. Se você queria escolher agregador e não o escolheu dentro de 10 seconds, você deve reinstalar a fim de voltar a este ponto em que você tem uma escolha de agregador.

- O sistema reinicialize automaticamente neste ponto para concluir a instalação. O primeiro login após um reboot requer uma mudança de senhas.

Quarto passso
A etapa inicial deve ser a configuração da rede, que deve ser feita localmente por meio da Interface de Linha de Comando (CLI) acessível através da porta serial ou do console do sistema. Nas etapas a seguir, você fornecerá vários parâmetros de rede para integrar o sistema Guardium® em seu ambiente, usando comandos CLI. Na sintaxe CLI, as variáveis são indicadas por colchetes emaranhados, por exemplo: < ip_enderes> Substitua cada variável com o valor adequado para sua rede e instalação. Não inclua os suportes.

Verificar instalação bem-sucedida.

...

Quinto passso
Verifique a instalação seguindo as seguintes etapas: - Login to CLI-ssh cli @ < ip do aparelho>

- Login em GUI- https://.:8443 (use o userid admin)

O primeiro login após um reboot exigirá uma mudança de senhas. Faça login na interface baseada na web do Guardium e acesse a ajuda online integrada para obter mais informações sobre qualquer uma das tarefas a seguir.

Para configurar um ambiente federado, configure um dos aparelhos como o gerente central e configure todos os outros aparelhos a serem gerenciados pelo gestor central. Use o comando CLI tipo unidade de loja para configurar o tipo de cada sistema Guardium.

Instalar chaves de licença: - Faça o login em seu sistema Guardium como o usuário admin .

- Verifique se o Tipo de Máquina exibido no banner Guardium está correto para o sistema que você está licenciando.

- Instalar uma licença base.

1. Navegar em Configuração > Ferramentas e Visualizações > Licença. 2. Na página Licença , insira a chave base para o seu sistema no campo Chave de Licença e clique em Aplicar para continuar. 3. A partir do diálogo Contrato de Licença , revise o contrato de licença associado com a chave base e clique em Aceitar quando estiver pronto para aceitar os termos. A interface Guardium atualizará automaticamente após aceitar o acordo, mas não haverá alteração na funcionalidade disponível após a instalação de uma chave de licença base.

- Instale uma ou mais licenças de anexação.

- Instalar patches de manutenção (se disponível)
