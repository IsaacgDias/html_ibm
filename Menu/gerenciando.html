Gerenciamento de acesso

O gerenciamento de acesso consiste em quatro tarefas: administração de contas, manutenção, monitoramento e revogação.

O Gerenciamento de Acesso é separado dos deveres de administração do sistema.

Há dois usuários predefinidos em um dispositivo Guardium ®: accessmgr e admin. accessmgr é o nome de usuário designado para o gerenciador de acesso. Por padrão, o gerenciador de acesso é o único usuário autorizado a gerenciar contas de usuários e funções de segurança. admin é o nome de usuário atribuído ao administrador do (primário) Guardium. Por padrão, o administrador não tem autoridade para gerenciar contas de usuário ou funções de segurança. O usuário admin tem um conjunto mais extenso de privilégios.

Compreender e gerenciar funções

Compreender Funções: Atribuir uma função a um usuário do Guardium para concedê-los privilégios de acesso específicos. Alguns exemplos de funções são: CLI, admin, accessmgr, CAS e usuário.

Gerenciando funções e permissões: Funções e permissões fornecem diferentes níveis de acesso aos usuários com base em suas tarefas de trabalho.

Gerenciamento de usuários

Gerenciando usuários: Use o Access Manager, designou o username accessmgr para adicionar contas de usuários, ativar ou desativar contas de usuários, importar membros do LDAP ou editar permissões de usuário. Abra o Navegador De Usuário e navegue pelas contas do usuário clicando em Acesso > Gerenciamento de Acesso > Navegador De Usuário

Criando um usuário que pode executar comandos GuardAPI: Crie um usuário que tenha as funções e autorizações apropriadas para executar comandos GuardAPI a partir da interface da linha de comandos (CLI).

Importando usuários do LDAP: Você pode importar Guardium definições de usuário a partir de um ou mais servidores LDAP configurando uma operação que importa o conjunto de usuários que precisam Guardium acesso.

Gerenciamento de dados e credenciais

Gerenciando credenciais do Guardium com o CyberArk: É possível usar o CyberArk para gerenciar suas credenciais do Guardium Para obter mais informações sobre esse aplicativo, acesse a página da web do CyberArk Marketplace e procure Guardium.

Segurança de Dados-Hierarquia do Usuário e Associações de Banco de Dados: Você pode usar recursos de segurança de dados para criar uma hierarquia de usuários e associar usuários a bancos de dados e servidores específicos. A segurança de dados do Guardium apresenta relatório sobre o qual os usuários acessaram quais informações, e garantem que apenas usuários específicos veem informações de que são responsáveis.
