Integração

Você pode integrar IBM® Guardium® com outros produtos: Configurar BIG-IP Application Security Manager (ASM) para se comunicar com o sistema Guardium: Use o ASM Big-IP (a partir de F5 Networks) juntamente com o monitoramento de atividade de banco de dados em tempo real do Guardium para resolver o problema de identidade propagação entre a aplicação web e camadas de servidor de aplicativos de banco de dados.

Integrações integradas: Você pode utilizar integrações disponíveis com os sistemas Guardium UNIX e Linux® .

Integrando com o Watson Knowledge Catalog para proteção de dados federados: É possível ativar a proteção de dados federados para integrar seu serviço IBM Watson™ Knowledge Catalog com o Guardium Data Protection para ajudar a assegurar que as regras de proteção de dados que são definidas no Watson Knowledge Catalog sejam impostas no nível da origem de dados de forma consistente nas origens de dados no escopo. Para ativar a proteção de dados federada, use os recursos de regravação de consulta do Guardium para criar e executar transformações (como mascaramento de dados) baseadas em regras do Watson Knowledge Catalog .

PIM Integration with Guardium DAM: O Privilegiado Information Management (PIM) ajuda as organizações a automatizar e rastrear o uso de identidades privilegiadas compartilhadas e monitorar o uso dessas identidades privilegiadas compartilhadas.

Configurando um sistema de chamado externo: Use um sistema de chamado externo como ServiceNow ou IBM Resilient para rastrear incidentes, problemas e tarefas descobertos pelo Guardium.

Integração do processador e do Guardium: O ambiente e o Guardium podem trabalhar juntos em um fluxo de informações de duas vias para ter as políticas de proteção de dados do Guardium atualizadas automaticamente e quase em tempo real em resposta a eventos de inteligência de segurança do produto do Ipdio.

OPTIM to Guardium Interface: Uma interface OPTIM para o Guardium, usando Protobuf (Universal Feed Agent), envia logs de atividade do Optim para o Guardium.

Combinando alertas em tempo real e análise de correlação com produtos SIEM: Distribui conhecimento contextual de padrões de atividade, estruturas e protocolos de banco de dados diretamente para o banco de dados de terceiros do sistema SIEM.

Mapeamento de CEF: O padrão de CEF de ArcSight define um conjunto de campos obrigatórios e um conjunto de campos opcionais.
